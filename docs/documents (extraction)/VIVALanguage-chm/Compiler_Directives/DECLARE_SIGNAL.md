# DECLARE SIGNAL

# 

# 

# 

________________________

### Syntax

DECLARE 
 SIGNAL signal name

signal name

Alphanumerical string, which indicates the name, assigned to the signal. 
 The signal name can be a maximum of 20 characters long.

__________________________

### Note

It is possible to define up to 1024 signals within a functional program.

__________________________

### See 
 also 

DECLARE CHANNEL

__________________________

### Examples

DECLARE SIGNAL example

Defines a signal. The defined signal can be handled like a channel name 
 or as a true variable. It is therefore possible to execute both arithmetic/logic 
 calculations and move it as if it were a channel.
