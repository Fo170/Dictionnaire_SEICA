# DECLARE PIN 

# 

# 

# 

________________________

### Syntax

DECLARE 
 PIN library name =channel name

library name

Name used in the library to describe the channels.

 channel 
 name 

Channel name defined in the DECLARE CHANNEL instruction.

__________________________

### Note

It is possible to define up to 1024 DECLARE PIN/ DECLARE CHANNEL per 
 functional program.

__________________________

### See 
 also 

DECLARE CHANNEL

__________________________

### Examples

DECLARE PIN example

Allows the association of 
 the channel name used in the library (P1, P2,...Pn) to the channel name 
 defined in the DECLARE CHANNEL instruction.
