# DECLARE SUBROUTINE

# 

# 

# 

________________________

### Syntax

DECLARE 
 SUBROUTINE subroutine name 1, subroutine name 2,

subroutine name 1,subroutine name 2,...

Alphanumerical string maximum 20 characters long that indicates the 
 name of the subroutine.

__________________________

### Note

Up to 2000 subroutines, functions and macros can be defined in a functional 
 program.
