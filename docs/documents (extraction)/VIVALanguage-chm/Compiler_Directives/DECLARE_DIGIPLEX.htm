DECLARE DIGIPLEX       

 

 

                                   

 

 

 

 

________________________

Syntax

 

DECLARE DIGIPLEX name=channel ;

 

name

name assigned to the specified channel

               channel

channel to the performing digital resource using DIGIPLEX technique

 

 

 

 

 

__________________________

Note

DIGIPLEX is a technique that provide digital resource by multiplexing

the 4 digital channel of the ACLAM through the analog scanner.

 

Declaring the channel it will use the DIGIPLEX technique even the the channel as digital capabilites

can be use for generate a backdriving since the ACLAM chennel as more power .
