~SET PW       

 

 

                                   

________________________

Syntax

 

~SET PWn options ;

 

n is a number between 1 and .... ; a number is assigned to each power supply through the ATE page of VIVA's System Configuration. (LINK)  

The options available for this instruction are listed here below.

 

ALL

Turns on and off all power supplies that were previously set with the PREPARE option.

V=value
