# ~APPLY CLOCK 

# 

# 

# 

________________________

### Syntax

~Apply clock 
 IL=value IH=value 
 OL=value OH=value 
 ACCV=value DELAY=value TO=signal_list

REF=signal 
 LABEL=name SYNC=name 
 TRIG=name USING=instrument 
 ; 

Parameters

Unit of measure

Mandatory

Options

Description

IL=value

[V]

No

Low level driver

IH=value

[V]

No

High level driver

OL=value

[V]

No

Low level sensor

OH=value

[V]

No

High level sensor

FREQ=value

[Hz]

Yes

Clock frequency

ACCV=value

[%] o [V]

No

Voltage accuracy required

DELAY=value

[s]

No

Delay between the trigger event and when the signal is generated by 
 the instrument

TO=signal_list

[-]

Yes

List of signals to which apply the voltage generator

REF=signal

[-]

No 

(default gnd)

Reference signal (commonly GND)

LABEL=name

[-]

No

Name to identify this clock application 

SYNC=name

[-]

No

Identification name of the trigger that will start the clock generator

TRIG=name

[-]

No

This trigger event will start execution of all previous instructions 
 that where synchronized with this trigger.

Example:

~ . SYNC=Pluto;

~.. TRIG=Pluto.;

The first instruction will start execution at the same time as the second 
 instruction, because it is synchronized to its trigger event.

USING=instrument

[-]

no

Name of the instrument you want to use
