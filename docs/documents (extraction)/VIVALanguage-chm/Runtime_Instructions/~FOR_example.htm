~FOR example

 

Example  1

 

!Executes the ~FOR loop until A is greater than zero and each time decrements the variable by 1  

 

~FOR A;

~WRITE “A= ”, A;

~ENDFOR;

 

 

 _____________________________________________________________________

 

 

Example 2

 

!Executes the ~FOR loop until A reaches the value of 10 incrementing the variable each time by 1  

 

~FOR A=1 TO 10;

~WRITE “A= ”, A;

~ENDFOR;

 

 

 _____________________________________________________________________

 

 

Example 3

 

!Executes the  ~FOR loop until  A reaches the value of 10 incrementing the variable each time by 2  

 

~FOR A=1 TO 10 STEP 2;

~WRITE “A= ”, A;

~ENDFOR;

 

 

 _____________________________________________________________________

 

 

Example 4

 

!Executes the ~FOR loop the first time with A=1, executes A=B+1 (A=5) executes the FOR loop the second time with A=5, executes A=B+1 !(A=6) executes the FOR loop the third time with A=6, executes A=B+1 (A=7) and so on until A reaches or exceeds the value 30

 

~CALC A=1;

~CALC B=1;

~FOR A=1 TO 30 DO A=B+3;

~WRITE “CICLO”, A;

~CALC B=B+1;

~ENDFOR;
