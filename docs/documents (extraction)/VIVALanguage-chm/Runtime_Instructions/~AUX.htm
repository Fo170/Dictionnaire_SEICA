~AUX       

 

 

 

________________________

Syntax

 

~AUX [*] [ON] [OFF] [INV]  modes (channel_list),[ modes, (channel_list)], etc.

 

 

[*]

Maintains the previous channel settings (specified in a previous ~AUX instruction.

 

ON

connect the channel frontend (only the first 4 channel by sequence) see ~CHOUT

 

OFF

disconnect the channel frontend (only the first 4 channel by sequence)

 

                   INV invert the function selected
