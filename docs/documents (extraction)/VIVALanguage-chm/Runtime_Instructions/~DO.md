### Syntax

~DO 
 num ;

...

...

...

~DONE ;

num

Number which indicates how many times the operations defined in the 
 ~do ~done cycle must be executed.

The ~DO instruction must be ended with a 
 ~DONE instruction.

### Examples

~DO example 

Executes the operations indicated in the 
 ~do ~done cycles the number of times indicated by num.
