### Syntax

~Apply IAC 
 V=value I=value 
 WAVE=option OFFSET=value FREQ=value 
 EVOLUTION=value ACCV=value ACCI=value 

 ACCF=value DELAY=value 
 COUPLING=option TO=signal_list REF=signal 
 LABEL=name SYNC=name 

 TRIG=name USING=instrument; 

Parameters

Unit of measure

Mandatory

Options

Description

V=value

[V]

Yes

Voltage programmed on instrument

I=value

[A]

No

Current programmed on instrument

WAVE=option

[-]

Yes

sine

rect

tri

pulse_pos

pulse_neg 

Sinusoidal waveform 

Square waveform

Triangular waveform

Positive impulse 

Negative impulse

OFFSET=value

[A]

No

Current offset for the signal

FREQ=value

[Hz]

Yes

Signal frequency

EVOLUTION=value

[-]

No

Number of full cycles of the waveform to perform

ACCV=value

[%] o [V]

No

Voltage accuracy required

ACCI=value

[%] o [A]

No

Current accuracy required

ACCF=value

[%] o [Hz]

No

Frequency accuracy required

DELAY=value

[s]

No

Delay between the trigger event and when the signal is generated by 
 the instrument

COUPLING=option

[-]

No

dc

ac

Coupling of the signal

TO=signal_list

[-]

Yes

List of signals to which apply the voltage generator

REF=signal

[-]

No 

(default gnd)

Reference signal (commonly GND)

LABEL=name

[-]

No

Name to identify this current application 

SYNC=name

[-]

No

Identification name of the trigger that will start the current generator

TRIG=name

[-]

No

This trigger event will start execution of all previous instructions 
 that where synchronized with this trigger.

Example:

~ . SYNC=Pluto;

~.. TRIG=Pluto.;

The first instruction will start execution at the same time as the second 
 instruction, because it is synchronized to its trigger event.

USING=instrument

[-]

no

Name of the instrument you want to use
