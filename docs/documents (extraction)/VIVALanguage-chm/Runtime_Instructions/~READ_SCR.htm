~READ_SCR

 

 

                                   

__________________________

Syntax

 

~READ_SCR value;

 

 

value

 Value read from the keyboard, this can be expressed as:

 

- a runtime variable runtime INTEGER type.

- a runtime variable runtime FLOAT type.

- a runtime variable runtime STRING type.

 

 

 

 

 

__________________________

Note

 

__________________________

See also                                                               

~READ_IEEE

 

__________________________

Examples

~READ_SCR example     

 

 

 

 

Interrupts program execution and waits for the operator to enter a value. A box appears on the static terminal, in the cursor position, and the operator can enter its value there. The number is stored in the specified variable.

Preprocessor *  Runtime :   Dynamic R    Static  p

 

Modules:  none
