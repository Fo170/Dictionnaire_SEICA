# 

# 

# 

________________________

### Syntax

~Pause 
 [value] ; 

value

Value displayed in the Pause Message box of the Static Terminal window 
 when the ~PAUSE instruction is executed.

The value can be expressed as:

 * 

a constant

 * 

a runtime variable INTEGER, FLOAT, STRING type

 * 

a string of characters enclosed by quotation marks 
 with maximum length of 80 characters

### Examples

~Pause example 

Inserts a pause in the execution of the 
 program, displaying the value if specified, and waits for a keyboard entry 
 by the operator.
