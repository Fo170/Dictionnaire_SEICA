### Syntax

~TEST_VBN 
 test_type 
 ; 

test_type=BYPASS

The components will all be set in bypass mode and a fixed string will 
 be sent through the boundary scan chain

test_type=IDCODE

Read the IDCODE of all the active components 

test_type=USERCODE

Read the USERCODE of all active components
