# ~READ_OPTOn 

# 

# 

# 

__________________________

### Syntax

~READ_OPTOn 
 variable; 

n

Whole number (from 1 to 4) indicating the USB module to read.

variable

INTEGER type runtime variable on which it is possible to read the opto-coupled 
 inputs.

The least significant bit of the variable corresponds to the opto-coupled 
 input 1.
