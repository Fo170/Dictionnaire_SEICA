#

________________________

### Syntax

~MSG 
 [value] ; 

value

Message displayed in the Pause 
 Message box of the Static Terminal window when the ~MSG instruction is 
 executed.

The value can be expressed as:

 * 

a constant

 * 

a runtime variable INTEGER, FLOAT, STRING type

 * 

a string of characters enclosed by quotation marks 
 with maximum 
 length of 80 characters
