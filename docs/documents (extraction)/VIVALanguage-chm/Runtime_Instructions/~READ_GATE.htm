~READ_GATE      

 

 

                                   

__________________________

Syntax

 

~READ_GATE pin;

 

 

 

pin

 

 

The frequency meter used for the measure is selected according to the specified pin (1 frequency meter every 8 pins).

~READ_FREQ example
