### Syntax

~Remove 
 label=name ; 

Parameters

Unit of measure

Mandatory

Options

Description

label=name

[-]

No

The command removes the previous apply or connect instruction that is 
 identified by this label.

-

[-]

No

ALL

This parameter removes all previous apply and connect instructions leaving 
 all the machine lines free.
