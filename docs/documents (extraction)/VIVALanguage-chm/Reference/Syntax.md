### Syntax

~KEYWORD 
 Parameter=value ; 

Parameter=value

Description of the parameter and default values

KEYWORD

Each command may only start with a recognized keyword

, = whitespace

Are valid separators for the parameter

;

Mandatory terminator.

Special 
 characters

@

identifies a command to be processed by the compiler 

~

identifies a static runtime command (optional in version 2.0.5)

%

identifies a compiled variable used in a runtime calculation or as value 
 of parameter 

#

identifies a file inclusion

*

stands for not in the pinlist

*

If at the beginning of the line it identifies a comment

!

Identifies a comment until the end of the line
