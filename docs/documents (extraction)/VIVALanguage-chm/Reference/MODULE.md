# MODULE-ENDMODULE 

# 

# 

# 

________________________

### Syntax

MODULE 
 module_name ; 

...

...

...

ENDMODULE 
 ; 

module_name

Alphanumerical string, which represents the name of the module. This 
 string can be maximum 9 characters long.

The MODULE instruction must always be ended 
 with an ENDMODULE instruction.

__________________________

### Note

 * 

It is possible to 
 declare up to 256 modules within one functional program.

 * 

after the keyword 
 MODULE all the declaration macros and include are reset and 
 must eventually redeclared
