# Installation 
 of drivers

To install new instrument drivers follow these simple steps:

 * 

On Viva's main toolbar click on System 
 Configuration 

 * 

Choose 
 the Installation page and click 
 on Driver Setting...

 * 

From the Driver 
 Setting page shows you the drivers already installed on your machine. 
 To install new drivers click on Install/Update 
 drivers and browse for the XML file containing the new drivers; 
 select it and press Open.
