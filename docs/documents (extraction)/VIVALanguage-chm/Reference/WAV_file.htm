This file contains the waveforms of the signals.

You can describe both digital and analog signals in this file.

 

 

 

________________________

Syntax

ANALOGIC WAVEFORMS

SIGNAL=name

CHANNEL=number

~pattern,KEY1=value,KEY2=value,...,END;

.......

~pattern,KEY1=value,....,END;

ENDSIGNAL

 

The options available as KEYWORDS are listed below:

 

MIN=value

Minimum value expected for the measure of this signal in this pattern

MAX=value

Maximum value expected for the measure of this signal in this pattern

VAL=value

 

TOLM=value

 

TOLP=value

 

READ=value

 

UNI=value

Unit of measure for the measurement; this field accepts the following units of measure:

- HZ (Hertz)

- V (Volts)

- A (Ampere)

These can be expressed in addition with :

- MEG

- M (milli)

- K (kilo)

- P (pico)

- U (micro)

- N (nano)
