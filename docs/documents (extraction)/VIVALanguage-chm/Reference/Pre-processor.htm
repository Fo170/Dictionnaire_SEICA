Pre-processor       

 

 

                                   

 

 

 

 

________________________

Syntax

 

@CALC   var1= expression ;

        @IF   condition  ; @ELSE;  @ENDIF;

        @FOR  var1=expression  TO value;  @ENDFOR;

          @MACRO  name  (par1 , par2 );   @ENDMACRO;

          @LABEL label;

 

 

var

any declared variable  

             expression

any calculation that involve only variable and constant.

 

 

 

 

__________________________

Note

The compiler as a preprocessor system  normally used in digital dynamic to generate

reapetive patterns, to group similar group of pattern in macro .

This facilities can also be used for any other pourpose.

 

To be keep in mind that variable and the control command are evaluate only by the processor

and the variable will retain their value only during compilation.

however the runtime variable retain their value only during execution.

 

It's not possible assign a runtime variable to a preprocessor variable.
