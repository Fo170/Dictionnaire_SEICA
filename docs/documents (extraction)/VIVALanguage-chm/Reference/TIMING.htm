TIMING       

 

 

                                   

 

 

 

 

________________________

Syntax

 

TIMING timing name

 

timing name

Alphanumerical string maximum 20 characters which defines the name of the timing.

 

 

 

 

 

 

__________________________

Note

The specified timing remains active also for the following patterns until another timing association is declared.

See also                                                               

Declaration instructions of the TIMING - ENDTIMING timing.
