### General Rules

### Each instruction line 
 must begin with the symbol #. At the bottom of the file you should list 
 the assignments of the signals to the number of channels.

________________________

### Syntax

### #type( 
 "name", 
 pin1, pin2, ..., pinN );

### 

type

Specifies the type of the component, the name with which the component 
 is described in the charLib 
 file.

name

Specifies the name of the component.

pin1,...,pinN

List of pins of the component.

The channels can be declared as INPUT, OUTPUT 
 and BIDIRECTIONAL.
