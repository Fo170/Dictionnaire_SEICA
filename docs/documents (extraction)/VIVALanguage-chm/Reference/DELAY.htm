DELAY       

 

 

                                   

________________________

Syntax

 

Delay time ;

 

 

time

Indicates the waiting time and can be expressed with the following letters:

P for ps (pico seconds)

N for ns (nano seconds)

U for ms (micro seconds)

M for ms (milli seconds)

S for s (seconds)

 

The accepted value is from 40ns to 30s.

Examples

Delay example     

 

 

 

 

Sets a delay time between the current pattern and the following one. During the delay time the sensor signals are not controlled, the channels associated to phases and repetitive clock are not generated.
