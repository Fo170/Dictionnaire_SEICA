### Syntax

NOSTEP 

__________________________

### Note

This instruction is used 
 when it is necessary to execute initialization or synchronization patterns 
 on the board under test before running the simulated program. These patterns, 
 in fact, must not be considered in the status file, which must start only 
 from the first simulated pattern.

### Examples

NOSTEP example

Disables the counter for the current pattern. The pattern number will 
 not appear in the list file File.pac and it will not be considered in 
 the status file of the simulator
