# NCLOCK 

# 

# 

# 

________________________

### Syntax

NCLOCK=value 
 ;

NCLOCK=AUTO 
 ;

CPP=value 
 ;

value

specified the duration in integer number of multiple of Period

AUTO

the number of repetition will be specified in the pattern option

default =1

__________________________

### Note

The result duration of the timing can't exceed the max duration of the 
 timing period.

CPP (Clock Per Pattern) is merely an alias of NCLOCK used in L200 emulation.
