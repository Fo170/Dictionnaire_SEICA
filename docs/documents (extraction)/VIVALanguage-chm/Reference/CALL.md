# CALL

# 

# 

# 

________________________

### Syntax

CALL 
 subroutine ; 

Subroutine

subroutine label with no parameter

__________________________

### Note

The user has to provide the trigger for 
 the digital burst if needed.

The subroutine should not contain any digital patterns.

__________________________

### See 
 also 

~CALL

__________________________

### Examples

Allows to execute a subroutine 
 during the execution of a digital dynamic burst.

The digital burst is launched and is immediately followed by the execution 
 of the subroutine. The program will continue when both the burst and the 
 subroutine are done.

Preprocessor Runtime : Dynamic R 
 Static 
 R

Modules: F40,DMAST,T964
