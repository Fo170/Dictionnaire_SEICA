LABEL       

 

 

                                   

 

 

 

 

________________________

Syntax

 

LABEL=label name

 

 

label name

Expresses the name of the LABEL in question. This can be an alphanumerical string maximum 20 characters. The underscore  “_” (underscore) is accepted in the label name while the use of special characters such as @ # $ % ~ etc... are not accepted.

 

 

__________________________

Note

Up to 1000 labels can be defined  in one functional program.
