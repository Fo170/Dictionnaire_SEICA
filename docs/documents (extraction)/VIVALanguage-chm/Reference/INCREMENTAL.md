### Syntax

INCREMENTAL

DEFAULT value: If the instruction 
 is not specified, the INCREMENTAL compilation mode is taken by default.

__________________________

### Note

The INCREMENTAL option is 
 very useful because it significantly decreases the time dedicated to programming 
 the channels; however it can be disadvantageous in the case of a program 
 with jumps. See EXAMPLE 3.
