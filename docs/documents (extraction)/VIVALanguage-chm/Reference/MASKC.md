### Examples

MASKC example 

Masks the measurement on all of the sensor channels of the pattern. 
 In case of error the global error flag is not activated, but only the 
 partial error flag is activated; therefore it is possible to execute a 
 measurement with the ONERROR and ONPASS instructions.
