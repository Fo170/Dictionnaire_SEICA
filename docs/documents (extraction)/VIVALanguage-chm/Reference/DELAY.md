### Syntax

Delay

time

Indicates the waiting time and can be expressed with the following letters: 

P for ps (pico seconds)

N for ns (nano seconds)

U for ms (micro seconds)

M for ms (milli seconds)

S for s (seconds)

The accepted value is from 40ns to 30s.
