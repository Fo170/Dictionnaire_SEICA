### Syntax

/label 
 name 

label name

Expresses the name of the LABEL. It can be an alphanumerical string 
 of up to 20 characters.

__________________________

### Note

There must not be a space between / and 
 label (Example: /messag1).

Up to 1000 labels can be defined in a functional 
 program
