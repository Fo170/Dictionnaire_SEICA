PULL

METHOD CLEAR

The method CLEAR disables the instrument PULL. It sends a reset command to set the instrument parameters to the default value. (See Tab. 51, in appendix A).

 

Syntax

VIVA LANGUAGE

 

~CLEAR PULL;

 

METHOD SET 

The method SET  selectsthe pull up/down/ext resistances on line L4. The resistances will be pull-up, pull-down or pull-ext according to the value of parameter IN, which defines the reference voltage value.

 

 

Syntax

VIVA LANGUAGE

 

~SET PULL       [VAL=value] SOURCE=option] [TIME_RELE=option]

Detail of parameters

 

VAL=option|value          [NONE|100|[1000|1K]|[10000|10K]|[100000|100K]|[1000000|1MEG],

default: NONE|0]

Defines the  resistive value of the pull-up/down resistance. The possible values of the parameter are:

NONE | 0             No pull-up/down resistance is connected to line L4.

100 | 1                A 100 pull-up/down resistance is connected to line L4.

1K|1000 | 2       A 1K pull-up/down resistance is connected to line L4.

10K|10000 | 3   A 10K pull-up/down resistance is connected to line L4.

100K|100000 | 4     A 100K pull-up/down resistance is connected to line L4.

1MEG|1000000 | 5  A 1M pull-up/down resistance is connected to line L4.

 

SOURCE=option|value   [NONE|L2|GND|VCC, default: NONE]

Defines the reference of the resistance defined with parameter VAL. The possible values of the parameter are:

NONE | 0             The resistance defined with parameter VAL is not connected to a reference point.

L2 | 1                  The resistance defined with parameter VAL is connected to line L2. (PULL-EXT)

GND | 2               The resistance defined with parameter VAL is connected to the system ground reference. (PULL-DOWN)

VCC | 3                The resistance defined with parameter VAL is connected to VCC voltage. (PULL-UP)

 

TIME_RELE=option|value         [ON|OFF, default: ON]

Defines if the output of the instruction ~SET PULLmust wait the switching time of the relays used to program the instrument. The possible values of the parameter are:

ON | 0                 The switching time of the relays is waited before the exit from the command

OFF | 1                After programming, the switching time of the relays is waited before the exit from the command

 

The parameter can be useful in case of sequential programming of different instruments: all the programmings but the last may be executed with the value of this parameter set to OFF, only the last one must be executed with this parameter set to ON.
