<expression>       

 

 

  _______________________

Syntax

      (value operator value [operator value] [operand (value operator value)])

 

                     

operator

expression

string expression

+

Addition

concatenated

-

Subtraction

 

*

Multiplication

get n characters

/

Division

convert to binary

^

Xor

convert to uppercase

&

And

 

|

Or

 

<

bit shift left

get n characters to the left

>

bit shift right

get n characters to the right

 

 

__________________________

Note

The expression must be included in parentheses.

 

In an expression referred to a compiler command, the runtime variable cannot be used.

 

In a runtime expression the variable must the preceded by the symbol %.
