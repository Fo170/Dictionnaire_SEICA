CLOCKF example

 

Example 1

 

System equipped with only F40 modules.

CLOCKF INT=20N;

TIMING TIM1;

STROBE 900N;

PERIOD 1000N;

ENDTIMING;

 

Example 2

System equipped with DHF and F40 modules.

CLOCK=100N;

CLOCKF=20N;

TIMING TIM1;

STROBE 900N;

PERIOD 1000N;

ENDTIMING;
