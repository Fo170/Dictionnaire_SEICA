# Présentation du Logiciel SEICA

Le logiciel VIVA est conçu pour gérer le développement et l'exécution de programmes de test sur des dispositifs électroniques à l'aide des systèmes de test Seica S.p.A. Il dispose d'une interface graphique conviviale et d'un ensemble d'outils logiciels pour guider l'utilisateur.

## Catégories d'Utilisateurs et Privilèges
Le logiciel est configuré pour accepter trois catégories d'utilisateurs différentes, chacune ayant des privilèges spécifiques :

| Catégorie d'utilisateur | Description des capacités |
| :--- | :--- |
| **Opérateur / Technicien** | Exécute le programme de test, importe une carte, exécute le diagnostic système et les procédures d'étalonnage, utilise la station de réparation. |
| **Développeur** | Possède tous les droits de l'opérateur, peut créer un programme de test et accéder aux fonctions de base de la configuration système. |
| **Administrateur** | Accès complet à toutes les fonctions du système. |

## Interface Principale
L'interface se compose de plusieurs éléments clés :
- **Barre d'outils supérieure :** Affiche les fonctions de création de programme sous forme de séquence (Import, Edit, Create, Multiply, Setup, Verify, Run).
- **Barre d'outils latérale droite :** Contient les utilitaires du gestionnaire système, du gestionnaire d'utilisateurs et du gestionnaire de programmes.
- **Section Favoris :** Barre d'accès rapide personnalisable.

## Utilitaires de Gestion
- **System Manager :** Sélection de programme, création, import, gestion de kit, sauvegarde/restauration système, auto-test, étalonnage, configuration système.
- **User Manager :** Gestion des utilisateurs VIVA, des modes d'authentification (Windows/Seica) et des permissions.
- **Program Manager :** Copie, suppression, export de données, gel/dégel de programme, environnement de programmation, statistiques, station de réparation.
